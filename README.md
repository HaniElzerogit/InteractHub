# Multi-Widget Web Platform 
An interactive educational project that brings together multiple web tools in a single, sleek interface. Built with core web technologies (HTML5, CSS3, JavaScript), it focuses on delivering a smooth user experience (UX) with a modern, elegant design.


 # Features
 
The platform includes 4 main interactive tabs:
1. Live Character Counter ✍️
A real‑time text editor with a countdown character counter (280 characters max). The counter changes color as you approach the limit — great for tweets or short messages.

2. Star Rating Widget ⭐
An interactive rating system with smooth visual feedback on hover and click. Perfect for collecting user feedback in a fun, visual way.

3. Web Power Tabs 💻
An educational tool that explains the difference between CSS (styling) and JS (interactivity). Click buttons to see how page elements change visually and behaviorally in real time.

4. Smart Alarm Clock ⏰
A real-time digital clock with a fully functional alarm system. Supports Snooze and Stop options, along with sound alerts and browser notifications.


# 🛠 Tech Stack
HTML5 – Clean, semantic structure

CSS3 – Modern styling with Flexbox and animations

JavaScript (Vanilla JS) – Core logic, timers, sounds, and interactivity

Bootstrap – Responsive design that works seamlessly on all screen sizes
